### To install

* mkdir Git
* cd Git
* git clone git@github.com:josefgrosch/sretools-library.git
* cd ~/Git/sretools-library/lib
* sudo pip install ./sretools
* cd ~/Git/sretools-library/docs
* pdoc --html sretools

### To use

* from sretools import < sub-module > as < alias >
